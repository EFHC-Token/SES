## Назначение книги

Книга создана как практический инженерный и экономический справочник по солнечной энергетике.

Материалы предназначены для:

- владельцев домов;
- фермерских хозяйств;
- бизнеса;
- промышленных объектов;
- EPC-подрядчиков;
- инженеров;
- проектировщиков;
- консультантов;
- специалистов по BESS и energy storage;
- пользователей, изучающих солнечную энергетику.
